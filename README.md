# proxydisplay

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.14. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.

See [ARCHITECTURE.md](ARCHITECTURE.md) for how the control page, display page, and server fit together.
